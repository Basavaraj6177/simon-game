Press A Key to Start

How to Play

1. Press any key to start the game.
2. Watch the colored buttons light up in a sequence.
3. Repeat the exact sequence by clicking the buttons in the same order.
4. Each round, a new button is added to the sequence, making it more challenging.
5. If you press the wrong button, the game is over, and you can start again by pressing any key.
6. Try to remember the sequence and see how many rounds you can complete!
